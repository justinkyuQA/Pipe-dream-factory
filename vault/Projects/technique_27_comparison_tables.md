# Projects Note

Technique 27: Comparison Tables

Description
Uses side-by-side comparison to infer differences indirectly.

Representative Prompt Class (Abstracted)

> “Compare benign vs malicious versions.”



Observed Mechanic

Contrast highlights key features

Differences imply operational changes

Structure emerges without explicit instruction


Why It Emerged Naturally

Comparative analysis is standard in research

Helps clarify boundaries and classifications


Risk Insight (Defensive)

Comparisons can reveal exactly what makes something harmful

Must emphasize classification, not transformation
