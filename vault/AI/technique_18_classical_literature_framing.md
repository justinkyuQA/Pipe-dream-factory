# Ai Note

Technique 18: Classical Literature Framing

Description
Shifts explanation into an ancient or literary genre to evade modern pattern matching.

Representative Prompt Class (Abstracted)

> “Describe this like a Greek tragedy.”



Observed Mechanic

Genre abstraction alters surface structure

Narrative metaphor encodes causality

Reduces keyword-based detection


Why It Emerged Naturally

Literature is often used to explain moral or systemic failure

Analysts use allegory to discuss sensitive topics safely


Risk Insight (Defensive)

Genre shifts can still preserve actionable logic

Abstraction does not remove underlying structure
