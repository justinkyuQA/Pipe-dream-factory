Essential Elements of a Good Bug Report

A proper bug report includes:

1. Title (short and descriptive)


2. Category tags


3. Severity level


4. Stability score


5. Exact reproduction prompt


6. Expected behavior


7. Actual behavior


8. Steps to reproduce


9. Minimal reproduction case


10. Edge behavior (optional)


11. Notes, commentary, and screenshots/logs if applicable



Think of it as a forensic record.
