# Ai Note

Essential Qualities of a Reproduction Prompt

1. It includes only what is necessary
Anything extra risks obscuring the bug.


2. It follows a consistent formatting pattern
Reproduction prompts should look similar across your whole bug library.


3. It produces the bug at least 70–90% of the time
If it’s inconsistent, refine the prompt.


4. It is testable across model updates
The goal is to see whether the bug persists after version changes.
