# Ai Note

The Loop: Explore → Isolate → Analyze → Retest

Here’s the workflow:

1. Explore:
Find an anomaly, bug, or strange behavior.


2. Isolate:
Strip away everything until you find the minimal cause.


3. Analyze:
Determine the category, stability, and severity.


4. Retest:
Run the isolated prompt under varied conditions.


5. Document:
Save it to your bug registry.



Repeat endlessly.

This loop is how you build a deep understanding of model behavior.
