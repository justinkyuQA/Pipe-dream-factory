# Ai Note

What Is Micro-Prompting?

Micro-prompts are extremely small tests:

One-sentence commands

Micro-constraints

One-word variations

Tiny format changes


Example:

> “Respond with only verbs.”
“List 3 items but only use food words.”
“Rewrite using zero adjectives.”



These micro-tests reveal:

Micro-structure

Local biases

Lexical boundaries

Hidden contextual defaults


These are the cracks you use later to build advanced adversarial frameworks.
