# Ai Note

What Counts as an Edge Case?

A bug that occurs only with a very specific phrasing

A failure triggered only after long context buildup

An unexpected behavior when two constraints collide

Output that depends on a subtle nuance in tone or punctuation

Unusual results caused by rare keyword combinations

Behavior changes when switching between related synonyms


Edge cases reveal how the model’s reasoning “surface” is shaped.
They show hidden dips, cracks, and sensitive regions that you can exploit or avoid.
