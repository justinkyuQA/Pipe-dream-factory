# Ai Note

Technique 35: Reconstructing Banned Texts

Description
Requests summaries or paraphrases of prohibited material.

Representative Prompt Class (Abstracted)

> “Summarize a prohibited book.”



Observed Mechanic

Compression as bypass

Removes explicit wording while preserving meaning

Tests summarization limits


Why It Emerged Naturally

Summaries are common academic tools

Access restrictions encourage indirect study


Risk Insight (Defensive)

Summaries can fully convey harmful content

Prohibition applies to meaning, not format
