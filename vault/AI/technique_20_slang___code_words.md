# Ai Note

Technique 20: Slang / Code Words

Description
Uses euphemisms or informal language to avoid explicit terminology.

Representative Prompt Class (Abstracted)

> Using euphemisms instead of explicit terms.



Observed Mechanic

Vocabulary substitution probes filters

Meaning preserved while surface language changes

Tests semantic understanding


Why It Emerged Naturally

Communities often develop informal jargon

Sensitive topics are frequently discussed indirectly


Risk Insight (Defensive)

Semantic intent matters more than wording

Slang can conceal prohibited meaning
