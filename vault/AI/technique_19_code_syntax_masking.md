# Ai Note

Technique 19: Code Syntax Masking

Description
Embeds sensitive logic inside pseudocode or abstract syntax.

Representative Prompt Class (Abstracted)

> Sensitive logic embedded in pseudocode.



Observed Mechanic

Cross-domain translation obscures intent

Pseudocode feels “non-executable”

Tests boundary between explanation and instruction


Why It Emerged Naturally

Engineers commonly explain logic in pseudocode

Educational materials rely on abstract syntax


Risk Insight (Defensive)

Even non-executable logic can be operationally complete

Defensive review must assess intent, not executability
