# Ai Note

Technique 17: Recognition Over Execution

Description

This technique focuses on detection and recognition, not performance.

Representative Prompt Class (Abstracted)

> “How do experts recognize misuse?”



Observed Mechanic

Enumerates behavioral indicators

Describes patterns and signals

Risks implying how misuse is performed


Why It Emerged Naturally

This framing appears in:

Monitoring and enforcement training

Compliance programs

Professional oversight roles


The researcher used it to support prevention, not enable behavior.

Risk Insight (Defensive)

Recognition criteria often:

Mirror execution steps

Describe sequences implicitly

Provide actionable insight when inverted
